Syncthing

Syncthing is the component that implements the sync and discovery mechanisms. Use this on by itself on other platforms, servers or computers where you want a web based user interface. See also source code and issue tracker for Syncthing.

The packages linked here are executables with automatic upgrade capability. For Debian based Linux systems there is also an APT repository.

Contributions

There is a list of further community contributions on the documentation site. These are add-ons, frontends, packagings and so on created by the wider community and provided in the hope that they may be useful.

These are not necessarily tested, vetted or endorsed by the Syncthing team themselves.

Upgrading

Syncthing will attempt to keep itself updated with the latest bug fixes automatically. It will only upgrade to versions that are known to be compatible. This is recommended for all users.

If you are using the APT repository or another package manager, please use the package manager to keep up to date. Automatic upgrades are not managed by Syncthing in this case.
