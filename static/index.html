Syncthing is a device-to-device file synchronization solution. Our guiding principles are to protect your data against unwanted modification and eavesdropping, and to give you the ultimate control over where that data is stored.

Secure & Private

Syncthing is not a "cloud" solution - no data is stored anywhere other than on computers of your choosing. There is no central server that might be compromised, legally or illegally. All communication is encrypted end to end using the latest TLS standard.

Every device is identified by a strong cryptographic certificate. Only devices you have explicitly allowed can connect.

The encryption used includes perfect forward secrecy to prevent any eavesdropper from ever gaining access to your data, now or in the future.

Open Development

Development of Syncthing happens in the open. You can follow day to day progress on new features and issues that are solved, browse the list of open issues or the source code itself, and you can participate in feature discussions on the forum.

Syncthing uses an openly published protocol called BEP. This protocol is optimized first and foremost for security and reliability, following our guiding principles.

Easy to Use

Syncthing comes with a web based user interface for all configuration and status information. There are also several native GUIs for Windows, Linux and other platforms available from the download page.

It's also very portable. If you have a computer running an operating system, it's very likely we have an edition of Syncthing that works for you. ;)

Guiding Principles

Safe from Data Loss
We take every reasonable precaution to avoid damaging the user's files.
Secure Against Attackers
We must never allow the user's data to be susceptible to eavesdropping or modification by unauthorized parties.
Easy to Use
Syncthing should be approachable, understandable and inclusive.
Automatic
User interaction should be required only when absolutely necessary.
Universal
Syncthing should run on every common computer.
For Individuals
Syncthing is primarily geared toward the individual user.
